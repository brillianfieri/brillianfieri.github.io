Reminder

Reminder is a tool for listing things that you need to be reminded of. This application enables users to create their own reminder lists and share them with others.

Summary

Key Features:
  • Registration and login using credentials.
  • Creation of reminder lists and interaction with reminder items.
  • Collaboration with others by sharing reminder lists through usernames.
  • Display of all reminders with date and time in the 'Scheduled' category.
  • Application of a dark mode scheme throughout the entire website.
  • Responsive design on various screen sizes.
Technologies Used:
  • Next.js 13 as the framework with the App directory.
  • MySQL as the database system.
  • Prisma as the database connector.
  • Tailwind CSS framework for styling.
  • Next-auth library for authentication.
  • Code snippets sourced from Flowbites, Stack Overflow, and Tutorialspoint websites.

In-Depth

1. Account and Authentication

This app uses Next-auth to control the authentication process. There are two account roles available:

  • User

    User is the default role of every new account. This role can only edit their account info and interact with the reminder features.

    User's Account Page

  • Admin

    Apart from using the reminder feature, this role can also manage the user list and view the logs. The owner will be prompted to register the admin user when the app runs for the first time. This role can make another user become admin through the Admin panel page.

    Admin Panel Page: This page contains three tabs: Account, Users, and Log. The 'Account' tab is similar to the user role's Account page, where the admin can modify their own account data. On the 'Users' tab, the admin can edit or delete users' accounts. In the 'Log' tab, the admin can review the activity logs across the app.

2. Reminder Features

In the app's sidebar, there are three main categories for the reminder list:

  • Your List

    In the 'Your List' category, all user-created lists are displayed.

    In this category, user can create their new list and add their reminder to it.

    The user can modify the list content using three buttons located at the top right corner of the app. The note and pencil icons will display the 'Edit' and 'Delete' buttons for each reminder item on the list.

    The 'Share' button allows the user to share the list with another user by entering the username.

  • Shared with you

    In the 'Shared with you' section, all lists shared with the user's account will be displayed. The user can collaborate on shared lists by adding, modifying, deleting, and marking reminder items as complete.

  • Scheduled

    This category serves as a summary of all reminders with deadlines, combining those from both the 'Your List' and 'Shared with you' categories. When the deadline passes for a reminder that hasn't been marked as completed, the reminder's color will change to red.
